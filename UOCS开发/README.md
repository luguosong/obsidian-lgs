# UOCS 开发知识库

## UOCS 是什么？

**UOCS**（绿建科技）是一个企业级综合性云平台，解决的核心问题是：让企业能够在一个统一的平台上完成电子文档的全生命周期管理——从 CA 证书签发、电子印章管理，到图纸查看/编辑/AI 智能审图，再到 PDF/OFD 文件的签名验证。它不是一个单体应用，而是由 **Spring Cloud 微服务后端 + React 前端 + 多个独立的文件查看器中间件** 组成的分布式系统。

本知识库将这些内容按子项目组织，帮助你从零理解整个平台的架构与实现。

## 新人阅读路线

如果你刚接触这个项目，建议按以下顺序阅读：

```
1. 架构总览/项目简介     → 建立全局概念模型
2. docker与部署/Docker部署架构 → 了解服务怎么跑起来的
3. upda-cloud/           → 理解后端微服务体系
4. drawing-cad/          → 深入最复杂的 CAD 子项目
5. 开发实践/             → 开始写代码前必读
```

每进入一个子项目，先读该目录下的 README.md，它会引导你阅读该领域的核心文档。
