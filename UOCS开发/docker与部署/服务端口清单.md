# 服务端口清单

> 端口分配约定（`170xx` 前端 / `180xx` 后端 / `181xx` 中间件 / `190xx` 数据库）见根目录 `docs/运维规范.md`。

## 数据存储

| 容器名 | 服务名 | 宿主机端口 | 容器端口 | 说明 |
|--------|--------|-----------|---------|------|
| uocs-mysql | mysql | `19002` | 3306 | MySQL 8.0，含 always-init-wrapper 自动重导入 |
| uocs-mongo | mongo | `19001` | 27017 | MongoDB 4.4 |
| uocs-redis | redis | `19003` | 6379 | Redis 7.2 |

## 中间件

| 容器名 | 服务名 | 宿主机端口 | 容器端口 | 说明 |
|--------|--------|-----------|---------|------|
| uocs-nacos | nacos | `18111` / `18110` | 8848 / 8080 | 配置+注册中心；gRPC: 18112→9848 |
| uocs-minio | minio | `18100` / `18101` | 8100 / 8101 | S3 API / 管理控制台 |
| uocs-rabbit-mq | rabbit-mq | `18120` / `18121` | 5672 / 15672 | AMQP / 管理控制台 |
| uocs-ollama | ollama | `18130` | 11434 | Embedding 模型服务（Drawing AI RAG） |
| uocs-qdrant | qdrant | `18140` / `18141` | 6333 / 6334 | 向量数据库 HTTP / gRPC（Drawing AI RAG） |

| uocs-server-gateway | server-gateway | `18010` | 8201 | Spring Cloud Gateway（对外入口） |
